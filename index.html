📖 Firebase Setup Guide

Follow these steps once — takes about 5 minutes. You only need to do this on one device; everyone else just needs the URL.

Step 1 — Create a Firebase project
  1. Go to console.firebase.google.com
  2. Click Add project → name it (e.g. OptimaED-SubCoverage)
  3. Click through the prompts → Create project
Step 2 — Create a Realtime Database
  1. In the left sidebar click Build → Realtime Database
  2. Click Create database
  3. Choose United States for location → Next
  4. Select Start in test mode → Enable
  5. Once the database is created, click the Rules tab
  6. Replace everything there with this and click Publish:
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
Step 3 — Get your web app config
  1. Click the gear icon ⚙ next to "Project Overview" → Project settings
  2. Scroll down to Your apps → click </> Web
  3. Give it any nickname (e.g. Sub Manager) → Register app
  4. Copy the entire firebaseConfig block shown on screen
Step 4 — Paste it here

Come back to Settings → Connect Firebase, paste the config, and click Connect & Sync. Share the HTML file or URL with your team — done!

⚠ The "test mode" rules expire after 30 days. The manual rules above (Step 2) are permanent and keep the database open to anyone with the link — which is what you want for this app.
